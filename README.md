# Akasha: A Ágora Digital de Rafael Neves

Este repositório contém o código-fonte e os recursos do meu portfólio pessoal, hospedado em [sr-rafael.github.io](https://sr-rafael.github.io ). Este projeto é mais do que um site; é um espaço para documentar minha jornada, meus projetos e minhas reflexões como um artesão digital.

---

## 🎯 Visão do Projeto

O objetivo da Ágora é ser um ponto central para apresentar minhas competências, projetos e filosofia de trabalho. Ele foi projetado para ser:

-   **Minimalista e Focado:** Uma interface limpa que coloca o conteúdo em primeiro lugar.
-   **Pessoal e Autêntico:** Construído do zero com HTML, CSS e JavaScript para refletir minha identidade como criador.
-   **Evolutivo:** Nascido dos fundamentos da web, com uma visão de futuro para ser reconstruído e servido pela robusta plataforma Elixir/Phoenix.
-   **Dinâmico:** Integrado com a API do GitHub para exibir informações atualizadas, como minha foto de perfil.

## 🛠️ Tecnologias Utilizadas

Este site foi construído utilizando a tríade fundamental da web, sem o uso de frameworks complexos, como um exercício de domínio dos fundamentos.

-   **Estrutura:** HTML5 Semântico
-   **Estilização:** CSS3 (com uso de Flexbox/Grid para layout)
-   **Interatividade:** JavaScript (Vanilla JS)
-   **Hospedagem:** GitHub Pages

## 🗺️ Roteiro de Desenvolvimento (Roadmap)

O desenvolvimento da Ágora seguirá as seguintes fases:

-   [x] **Fase 1: Estrutura (HTML):** Definição da fundação semântica de todas as páginas.
-   [ ] **Fase 2: Estilização (CSS):** Implementação da identidade visual, layout responsivo e temas (claro/escuro).
-   [ ] **Fase 3: Interatividade (JS):** Adição de funcionalidades, como o botão de troca de tema.
-   [ ] **Fase 4: Conteúdo:** Preenchimento de todas as seções com os detalhes dos projetos e habilidades.
